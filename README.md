# Pori

A command line toos for Bitbucket to create repository.

## Installation

Add this line to your application's Gemfile:

    gem 'pori'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install pori

## Usage

    $ cd path/to/local-git-directory
    $ pori
